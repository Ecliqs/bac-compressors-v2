# **App Name**: BAC Air Solutions

## Core Features:

- Dynamic Content Display: Showcase all static content sections: Hero, Trust & Credibility, Education, Industries Served, Manufacturing Strength, and 'Why Choose Us'.
- Product Information Hub: Present detailed sections for both Reciprocating and Screw Air Compressors, including descriptions, key features, and dedicated Call-to-Action buttons.
- Multi-Point Lead Capture Forms: Integrate the primary 'Get Best Price' form and various 'Request Price/Quote' buttons to collect user inquiries efficiently.
- Sticky Call-to-Action (CTA): Implement a persistent, visible CTA button that remains accessible as the user navigates the page, encouraging conversion.
- Customer Testimonial Slider: Display a carousel of customer testimonials to enhance social proof and build trust with prospective clients.
- Mobile-Responsive Layout: Ensure the entire landing page is fully optimized and displays perfectly across all desktop, tablet, and mobile devices.
- Intelligent Compressor Advisor Tool: An AI-powered tool that provides initial compressor type recommendations based on specific requirements gathered from the lead capture form (e.g., industry, air demand).

## Style Guidelines:

- Primary color: #1F3D7A. A deep, professional blue representing reliability and engineering precision, suitable for key elements and text.
- Background color: #F0F1F5. A very light, cool gray that provides a clean, neutral canvas for industrial content.
- Accent color: #26BBD9. A vibrant, yet industrial cyan chosen to highlight calls-to-action and important interactive elements, creating strong visual contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and highly legible look across all content, reflecting the app's industrial precision.
- Utilize robust, clean-lined icons in a monochromatic style, particularly for industry representation and feature highlights, to reinforce the industrial aesthetic.
- Implement a conversion-optimized layout with clear section hierarchies, ample whitespace, and strategic placement of multiple lead forms and prominent CTAs.
- Employ subtle, performance-friendly animations such as hover effects on interactive elements and smooth transitions for section reveals, ensuring fast loading and a polished user experience.
